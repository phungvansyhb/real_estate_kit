# Page SEO Checklist

Dùng checklist này trước khi kết thúc một task SEO.

## Indexability

- [ ] Page này thực sự nên được index
- [ ] Nếu không nên index, đã set `robots` phù hợp
- [ ] Canonical URL đã trỏ về URL sạch, duy nhất

## Metadata

- [ ] Có `title` riêng cho page
- [ ] Có `description` riêng, không copy-paste toàn site
- [ ] Có `openGraph` tối thiểu: title, description, url
- [ ] Có `twitter` card phù hợp

## Content

- [ ] Chỉ có một `h1`
- [ ] Heading `h2/h3` theo hierarchy hợp lý
- [ ] Nội dung chính xuất hiện sớm và khớp intent tìm kiếm
- [ ] CTA rõ ràng nhưng không lấn át nội dung chính

## Listing page

- [ ] Metadata được sinh từ dữ liệu listing
- [ ] Ảnh cover phù hợp cho OG
- [ ] Mô tả không bị trùng hàng loạt giữa các listing
- [ ] Có thông tin giá, diện tích, vị trí nếu dữ liệu tồn tại

## Technical

- [ ] Dùng Metadata API thay vì meta tag thủ công nếu có thể
- [ ] Không đưa route private vào sitemap
- [ ] Ảnh public có thể truy cập được từ bot/social crawler
- [ ] Không thêm structured data giả hoặc đoán dữ liệu
